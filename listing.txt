1 Manifest
==========

  - Zippert-Tristan-literate_writeup.pdf: Literate writeup PDF
    containing code explanation, code, and code measurement.
  - Zippert-Tristan-literate_code.lisp: Source Lisp code extracted from
    the literate writeup
  - Zippert-Tristan-example_output.txt: REPL buffer showcasing example
    input and output of the algorithm
  - README.txt: This file


2 Running
=========

  - 'Zippert-Tristan-literate_code.lisp' can be run through SBCL with
    the following command: `sbcl --load
    Zippert-Tristan-literate_code.lisp'
